# 🍰 Cake Shop Website

A modern cake shop website built with **HTML5** and **CSS3**.

This project showcases a clean and elegant bakery website, focusing on modern layout techniques using **CSS Flexbox** and **CSS Grid** while creating a visually appealing user interface.

---

## ✨ Features

- 🍰 Elegant Cake Shop Landing Page
- 🎨 Clean & Modern User Interface
- 📐 Built with CSS Flexbox & CSS Grid
- 🧩 Semantic HTML5 Structure
- ✨ Clean, Readable & Maintainable Code
